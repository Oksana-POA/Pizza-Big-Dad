Авторизация

Цель: авторизоваться на сайте
Акторы: пользователь, система.
Основной сценарий:
1.	Пользователь в корзине нажимает «Оформить заказ»
2.	Система показывает экранную форму авторизации.
3.	Пользователь вводит номер телефона.
4.	Пользователь нажимает кнопку «Получить код»
5.	Система отправляет запрос в сервис нотификации
6.	Система ведет обратный отчет времени от 45 сек для ввода кода
7.	Пользователь получает смс с кодом.
8.	Пользователь вводит код на форму авторизации
9.	Система переводит на форму доставки
Альтернативный сценарий
1.	Пользователь вводит номер телефона
2.	Система отправляет запрос в сервис нотификации
3.	Пользователь не вводит код в течении 45 секунд
4.	Система активирует кнопку «отправить код повторно».
5.	Пользователь нажимает кнопку «отправить код повторно».
6.	Система отправляет запрос в сервис нотификации
7.	Пользователь получает смс с кодом.
8.	Пользователь вводит код на форму авторизации
9.	Система переводит на форму доставки
Исключения:
Некорректный номер
1.	Пользователь вводит номер телефона
2.	Система отправляет запрос в сервис нотификации
3.	Система получает ошибку от сервиса нотификации
4.	Система выдает ошибку пользователю «Номер не существует»
